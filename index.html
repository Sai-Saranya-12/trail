Introduction to hr tag

For in detail explanation about hr element, please refer devdocs


hr element is mainly used to get horizontal line. There are many attributes availabe in hr tag.

  • Align: Used to emphasize if the line should be right aligned or left aligned.
  • Size: Determines the height of the line in pixels
  • noshade: determines it should have any color
  • Color: determines the color of the horizontal line
  • Width: determines the width of the horizontal line
